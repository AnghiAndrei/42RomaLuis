https://localhost:8000/users/login
Method post
Campi Json aspettati: email, password
Campi Json ritornati: key
Cosa fa: verifica se le credenziali esistono nel db e se si devi inviare una email con il codice: OTP e ritornare la chiave per il codice OTP
Codici ritornati:
    200 tutto ok
    204 credenziali sbaliate
    500 altri errori

https://localhost:8000/g1/save_game
Method post
Campi Json aspettati: nomep1, nomep2, nomep3, nomep4, esito, pp1, pp2, pp3, pp4, gioco (Nota: ti verra passato il JWT nella autorizzazioni)
Campi Json ritornati: e
Cosa fa: carica la partita nel bd
Codici ritornati:
    200 tutto ok
	401 token invalido
	404 valori mancanti
    500 altri errori

https://localhost:8000/g1/get_games
Method get
Campi Json aspettati: NULL (Nota: ti verra passato il JWT nella autorizzazioni)
Campi Json ritornati: totalewin, totalelose, totaledraw,totalegame, {(array da max 20 elementi, in ordine dal piu recente indietro) nomep2, nomep3, nomep4, esito, data, punteggio, gioco, tipo}
Cosa fa: ritorna le stat e le ultime partire
Codici ritornati:
    200 tutto ok
    400 se non ci sono dati
    500 altri errori
